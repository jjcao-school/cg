# Computer Graphics - 2021 (DLUT)
## 0. Introduction
授课32学时（上机24学时）.内容涵盖光栅化rasterization和光线追踪等渲染算法，3D几何建模的数学概念和算法，计算机动画算法，应用为主，理论为辅，大家需要多动手实验，积极解决实际问题。通过这门课程的学习，你将会学习到计算机视觉和计算机图形学的基本知识，学习和掌握一定的编程技能。
